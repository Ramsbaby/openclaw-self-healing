# Memory 구조

## 디렉토리 레이아웃

```
memory/
  ├── README.md (이 파일)
  ├── YYYY-MM-DD.md (일일 로그)
  ├── profiles/
  │   └── owner.md (정우님 프로필 - 영구)
  ├── knowledge/
  │   └── openclaw-system.md (시스템 지식)
  └── projects/
      └── sk-dnd-iot.md (프로젝트 정보)
```

## 사용 가이드

### profiles/
영구적인 프로필 정보
- owner.md: 정우님 경력, 기술 스택, 철학

### knowledge/
도메인별 지식
- openclaw-system.md: 시스템 설정, 크론, MCP, 스킬

### projects/
프로젝트별 문서
- sk-dnd-iot.md: 현재 프로젝트 정보

### 일일 로그 (YYYY-MM-DD.md)
날짜별 활동 기록 (기존 방식 유지)

## 검색 최적화
모든 Markdown 파일은 memorySearch로 검색 가능
